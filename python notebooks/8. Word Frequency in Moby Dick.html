Project Description¶

In this workspace, we will scrape the novel Moby Dick from the website Project Gutenberg, which offers a large collection of free books. We use the Python requests package to download the text of the novel. Then, we extract words from the downloaded web data using the BeautifulSoup library.

After collecting the words, we analyze their frequency and distribution using the Natural Language Toolkit (nltk) and the Counter class from Python's collections module. This helps us understand how often different words appear in the novel.

The data science pipeline we build here can be applied to any novel available on Project Gutenberg. This means we can visualize and study the word frequency patterns of many different books by following the same steps.

What are the most frequent words in Herman Melville's novel Moby Dick, and how often do they occur?¶

# Create a BeautifulSoup object from the HTML
html_soup = BeautifulSoup(html, "html.parser")

# Get the text out of the soup
moby_text = html_soup.get_text()

# Create a tokenizer
tokenizer = nltk.tokenize.RegexpTokenizer('\w+')

# Tokenize the text
tokens = tokenizer.tokenize(moby_text)

# Create a list called words containing all tokens transformed to lowercase
words = [token.lower() for token in tokens]

# Print out the first eight words
words[:8]
